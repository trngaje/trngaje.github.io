miyoo mini + generates keyboard value when press each button.
button value is below.

Button | ev key value
------ | --------
B | KEY_LEFTCTRL
A | KEY_SPACE
Y | KEY_ALT
X | KEY_SHIFT
L1 | KEY_E
R1 | KEY_T
L2 | KEY_TAB
R2 | KEY_SPACE
up | KEY_UP
down | KEY_DOWN
left | KEY_LEFT
right | KEY_RIGHT
select | KEY_RIGHTCTRL
start | KEY_ENTER
volume up | KEY_VOLUMEUP
volume down | KEY_VOLUMEDOWN
menu | KEY_ESC
